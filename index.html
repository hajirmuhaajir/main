// Tentang Saya

Siapa Hajir?

Fotografer, musisi, dan developer yang berkarya lewat seni digital.

Hajir Muhaajir

Fotografer · Musisi · Developer

Fotografer Musisi Developer Bogor

Saya Muhaajir — seorang pencinta fotografi dan musik dari Bogor. Setiap foto adalah cerita, setiap nada adalah emosi yang tak terucap.

Melalui karya, saya ingin berbagi perspektif dan menginspirasi siapapun yang menyaksikannya. Bergabunglah dalam perjalanan kreatif ini.

Fotografi, musik, dan seni digital menjadi bahasa sehari-hari saya. Saya juga membangun tools browser-based yang ringan dan privasi-first.

Tools yang Saya Buat

Setiap tool dibangun ringan, cepat, privasi-first, dan tanpa server. Semua berjalan 100% di browser.

LRC Generator
Stable

HajirSync

Generator lirik LRC berbasis browser dengan visualizer gelombang audio Web Audio API. Sinkronisasi lirik real-time, ekspor file LRC instan, antarmuka bilingual (ID/EN).

Web Audio API LRC Export Bilingual UI
Buka Tool
Password Manager
Stable

Muhaajir Vault

Vault password premium dengan generator karakter hingga 50+, strength meter visual, riwayat password, dan selektor bahasa 13+ bahasa. Data hanya di browser kamu.

Password Gen Strength Meter 13 Languages
Buka Tool
Metadata Stripper
Beta

Muhaajir Scrub

Hapus metadata EXIF/GPS dari foto dan PDF sebelum diunggah. Privasi total — semua proses berjalan 100% di browser, tidak ada data yang dikirim ke server.

EXIF Cleaner GPS Remover PDF Support
